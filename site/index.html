Here I will be posting some HTML/CSS/Javascript files that are solutions to the problems presented to us within the course, posted for peer review; and at some point I would also like to create some type of updated information for the course, including getting through the Git setup headache a bit more smoothly.

Note to instructor Yaakov that, while in my opinion GitHub is very good and useful for uploading and hosting these pages for students, it is important to remain in-vogue with the more popular options available on the Web for public uploads from students. A cloud storage box (Google Drive or Dropbox) is an example of something just a tiny bit more user-friendly for beginners, and while they aren't always free for huge amounts of data and files, some of them have free options that can give your free account space of up to 100 GiB.


Also, in my honest opinion, VS Code is a bit more popular and user-friendly than the simple text-editors, and its intuitive plugin set make it very easy to format the code we are learning without having to pay too much attention to document formatting techniques such as tab trees. It's also free and users can easily add plugins that will help them code in multiple languages such as Python, C, C++, and Java, without having to download multiple editors and compilers.
